::: details 1. React / Vue 项目时为什么要在列表组件中写 key，其作用是什么？


```
key是给每一个vnode的唯一id,可以依靠key,更准确, 更快的拿到oldVnode中对应的vnode节点。

1. 生成唯一的索引，diff算法方便查找
2. 避免复用

```
